# geoMaster | Master your worldly Knowledge

A retro pixel art geography quiz game with 4 game modes, global leaderboards, and PWA support.

## Features
- 4 Game Modes: Country, Flag, Capital, Guesser
- Real-time scoring with penalty system
- Global leaderboards via Supabase
- Light/Dark theme toggle
- 8-bit retro audio effects
- PWA support for offline play
- WCAG accessibility compliant

## Tech Stack
- HTML
- CSS
- Javascript
